# 备课铃响之前 · Before the Bell

一款**教师资格考试 × 校园**主题的剧情向视觉小说原型(prototype)。你将作为一名新教师，在虚构的「星澜高中」里一边融入校园、一边在剧情中自然地学习教师资格考试(综合素质 / 教育知识与能力)的真实考点。

> 全部人物、学校、情节均为**虚构**。考点内容用于学习演示。

## 在线试玩

GitHub Pages 部署后即可直接在浏览器游玩(见仓库 Settings → Pages 给出的链接)。

## 特色

- 📖 **剧情中学考点**：知识点包裹在有代入感的师生 / 同事剧情里，不是干背。
- 🎴 **数据驱动**：章节、题库、知识图谱均为 JSON，易扩展。
- 🧠 **间隔重复 + 每日一关**：循证记忆机制，巩固易忘考点。
- 📝 **月考 / 模考**：按真实笔试结构组卷，客观题判分 + 主观题自评。
- 🎨 自包含设计系统、章节入场动画，零外部 UI 依赖。

## 技术栈

React 19 · TypeScript · Vite 7

## 本地运行

```bash
npm install
npm run dev      # 开发预览
npm run build    # 生产构建(输出 dist/)
```
